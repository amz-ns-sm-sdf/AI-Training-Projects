# Project 3: Chat with Google Login & Thread Management

## Description
Add Google OAuth login and chat thread management with auto-generated thread names.

## Features
- Google OAuth login
- Create/update/delete chat threads
- Auto-assign names to threads
- Load threads on login
- Built on Project 2

## Tech Stack
- **Frontend**: HTML/CSS/JavaScript with Google Auth
- **Backend**: Python with Flask
- **Database**: PostgreSQL
- **Auth**: Google OAuth 2.0

## Setup Instructions
1. Create Google OAuth credentials
2. Add Google Client ID to `.env`
3. Install dependencies: `pip install -r requirements.txt`
4. Run: `python app.py`
