# Project 1: Simple Chatbot with Gemini LLM

## Description
Create a simple chatbot that talks to an LLM (Google Gemini) and provides answers.

## Features
- UI with any framework (React/Vue/Angular)
- Python backend using Langchain framework
- Gemini API integration

## Tech Stack
- **Frontend**: HTML/CSS/JavaScript (or framework of choice)
- **Backend**: Python with Langchain
- **LLM**: Google Gemini

## Setup Instructions
1. Backend Setup:
   - Install Python dependencies: `pip install -r requirements.txt`
   - Add your Gemini API key to `.env`
   - Run: `python app.py`

2. Frontend Setup:
   - Install dependencies: `npm install`
   - Run: `npm start`
